NF Captcha adds Really Simple CAPTCHA element for human check.

== Description ==

NF Captcha adds really simple captcha elements to the Elements' block. It also includes labels and custom classes for the element to allow for more styling.
It ist a good alternative, if you don't want use reCAPTCHA or Anti-Spam-Question.

This plugin requires the Ninja Forms and Really Simple CAPTCHA plugin.

== Installation ==

1. Upload 'nf-captcha.php' to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
